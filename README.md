# gitlab_new

Automation scripts for account creation and Codeanywhere session management.

## Clone

```bash
git clone https://github.com/hasnaouiyacine59-wq/gitlab_new.git
```

## Scripts

- `lab_get.py` — Automated Atlassian account creation + Codeanywhere workspace setup
- `ca_open.py` — Open a saved Codeanywhere session using cookies (`-s <session_number>`)

## Install

```bash
pip install -r requirements.txt
playwright install chrome
```
